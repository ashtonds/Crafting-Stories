# ✨ Crafting Stories

A modern storytelling and creative portfolio website focused on preserving memories through photo restoration, colorization, digitization, and cinematic editing.

Crafting Stories combines elegant design, smooth interactions, and immersive visuals to showcase creative work in a memorable way.

## 🌐 Live Demo

🔗 https://ashtonds.github.io/Crafting-Stories/

---

## ✨ Features

🎨 Elegant premium UI design

📱 Fully responsive for desktop and mobile

🖼 Interactive Before/After comparison sliders

📷 Photo restoration showcase

🌈 Black & White → Color transformation

🎞 Negative film → digital conversion preview

🎬 Video & reels editing services section

⚡ Smooth animations and scroll reveal effects

🖱 Custom animated cursor

📞 Direct WhatsApp contact integration

---

- CSS3

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/ashtonds/Crafting-Stories.git
```

Move into the folder:

```bash
cd Crafting-Stories
```

Open the project:

```bash
index.html
```

Or run with VS Code Live Server.

## 💡 Main Sections

### Hero Section
Clean landing page with animated elements and call-to-actions.

### Services
- Photo Restoration
- B&W → Colour
- Negative Film Conversion
- Video Editing
- Reels Editing
- Travel Videos

### Before/After Comparison
Interactive sliders allowing users to drag and compare transformations.

Transparent pricing cards with service breakdowns.
